Automated Forex Trading Bot – Key Functionalities

1. Order Placement (Pending Orders)

  Place Buy Limit and Sell Limit orders automatically at specified prices and volumes.

2.  Profit & Volume Analysis

  Calculate total profit for a symbol.

  Track profit for buy-only and sell-only positions.

  Monitor total trade volume.

 3. Margin Calculations

  Compute margin used for buy positions and sell positions.

  Evaluate percentage profit relative to margin for smarter risk management.

4.  Position Management

  Close a single position or all positions for a given symbol.

  Manage pending orders: delete specific orders or clear all pending orders.

5.  Grid Trading Automation

  Automatically places a grid of buy and sell limit orders around current market prices.

  Buys below market, sells above market — perfect for range trading strategies.

6.  Continuous Monitoring & Automation

  Continuously monitors positions and margin.

  Calculates percentage profit in real-time.

  Automatically closes positions when a profit target is reached.

  Removes pending orders when no positions are open.


Technologies Used:
Python | MetaTrader 5 | pandas | threading | Tkinter (for GUI)
